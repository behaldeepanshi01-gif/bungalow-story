## About

A single-page narrative website built for Bungalow NYC (Michelin-star restaurant by Vikas Khanna). Features smooth scroll animations using Intersection Observer API, custom SVG architectural illustrations, and an emerald/gold design system that mirrors the restaurant's aesthetic.

## Features

- Scroll-triggered section animations (Intersection Observer API)
- Custom inline SVG illustrations for architectural elements
- Navigation dot indicators for section tracking
- Responsive design for mobile and desktop
- CSS custom properties for consistent theming
